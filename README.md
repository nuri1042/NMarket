# NMarket 프로젝트

- 프로젝트 주제 : Next.js 를 이용한 쇼핑몰 제작

- Programming Languages : Javvscript, Typescript
- Frontend Development : Next.js

- 상태관리 : React-Redux, Redux, axios
- 
- Styled-Components, Ant Design
